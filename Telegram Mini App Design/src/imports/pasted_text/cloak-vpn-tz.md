1. Общая информация

Название проекта: Cloak VPN Mini App
Платформа: Telegram Mini App (Telegram WebApp SDK)
Цель: Дать пользователю быстрый доступ к своей VPN-подписке, ссылке (vless) и удобному импорту в конфигураторы (Happ, V2RayTun и др.).
Основной сценарий: за 10–15 секунд пользователь получает ссылку / QR-код и подключается к VPN.
Авторизация: автоматическая по Telegram ID (tg.initDataUnsafe.user.id).
Ограничения: максимум 1 активная подписка на аккаунт.
Пробный период: 3 дня — только один раз за всю историю аккаунта (повторно только через промокоды).
Ссылка: одна универсальная vless://… (не меняется до конца подписки).
После истечения подписки: ссылка и вся карточка полностью скрываются сразу.

5. Пользовательские сценарии (основные flows)

Первый запуск
Авто-активация пробного периода 3 дня → редирект на Главную → пользователь видит «Пробный период» и кнопку «Подключиться».
Обычное подключение
Главная → «Подключиться» → выбор ОС → выбор приложения → «Открыть в приложении» или «Другое» (QR в центре).
Продление
Подписки → «Продлить» → выбор срока/устройств → оплата → webhook → Главная.
Промокод
Настройки → ввод промокода → тост «Промокод применён!» → если дал пробный период — поведение как при первом запуске.
Истечение подписки
При следующем открытии миниаппа: сразу состояние «Нет ключей» (ссылка скрыта полностью).

6. Технические требования к бэкенду
Нужные эндпоинты:

GET /user/sub — текущая подписка + ссылка + expires_at + devices + trial_flag
POST /trial/activate — активировать пробный период (только один раз)
POST /payment/create — создание платёжной ссылки
POST /promo/apply — применить промокод
GET /user/ref — реферальные данные

Webhook от платежной системы — обязателен (мгновенное обновление подписки).
7. Дополнительные требования

Формат остатка времени:
«18 дней 3 часа»
Если < 1 дня: «Осталось 14 часов 45 минут»

Поле промокода — только в Настройках.
После истечения подписки — ссылка исчезает сразу.
Все действия с ссылкой (копировать, открыть, QR) работают без перезагрузки страницы.
